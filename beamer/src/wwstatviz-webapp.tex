  \begin{itemize}

    \item handy to launch small applications or web page
  \end{itemize}

The following code aims at displaying "Hello World!" on a web page: 

\begin{lstlisting}[language=Python]
  from flask import Flask
  app = Flask(__name__)

  @app.route("/")
  def hello():
      return "Hello World"


  if __name__ == "__main__":
      app.run(debug=False)
\end{lstlisting}
